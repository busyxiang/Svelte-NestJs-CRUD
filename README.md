# Svelte-NestJs-CRUD

## Run Application

You would need two terminals in order to run both frontend and backend at the same time in this projects.

### Backend Terminal

```bash
$ cd ./nestjs-backend
$ npm run start:dev
```

### Frontend Terminal

```bash
$ cd ./svelte-frontend
$ npm run dev
```
